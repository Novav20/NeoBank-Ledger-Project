# NFR-019: Provide Independently Auditable Quorum Certificates per Block

## Statement
The Ledger system shall attach quorum-certified evidence to each block so that every block is independently auditable.

## Rationale
[[BPA_Report#4.2 Consolidated NFR Registry|BPA NFR-AUDIT-01]] defines the audit target as $2F+1$ QC per block and independently auditable. [[barger_2021]] shows $2F+1$ signatures in block metadata for external verification, [[praveen_2024]] uses quorum certificates and monitors to validate certificates, and [[benedetti_2022]] treats the block solution as the quorum certificate. This makes the auditability requirement directly traceable without introducing a gap.

## Acceptance Criteria
- [ ] Each block carries quorum-certified evidence suitable for external audit.
- [ ] An independent observer can verify the block evidence without participating in consensus.
- [ ] The audit trail preserves the evidence needed to reconstruct the block's certification state.

## Notes
This requirement covers block-level audit evidence only; it does not redefine finality or settlement semantics.
