# NFR-007: Use ISO 20022-Compliant Inter-Ledger Messaging

## Statement
The Ledger system shall use ISO 20022-compliant message formats for inter-ledger communication.

## Rationale
[[BPA_Report#4.2 Consolidated NFR Registry|BPA NFR-COMP-01]] defines the compliance target as ISO 20022 inter-ledger messaging. [[chuen_2017]] identifies ISO 20022 as the relevant financial messaging standard, and [[sonnino_2021]] also cites ISO 20022 in the context of real-time domestic payments. This makes the messaging requirement directly traceable and standards-aligned.

## Acceptance Criteria
- [ ] Inter-ledger messages conform to the ISO 20022 message structure used by the project.
- [ ] Message validation rejects non-compliant inter-ledger payloads.
- [ ] The compliance rule is observable in schema and interoperability tests.

## Notes
This requirement is about message format and standards conformance, not about settlement timing or audit evidence.
