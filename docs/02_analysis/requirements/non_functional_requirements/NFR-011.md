# NFR-011: Maintain a Post-Quantum Cryptographic Migration Roadmap

## Statement
The Ledger system shall maintain a documented roadmap for migrating to post-quantum cryptography.

## Rationale
[[BPA_Report#4.2 Consolidated NFR Registry|BPA NFR-COMP-05]] defines the compliance target as a post-quantum cryptographic migration roadmap. [[wang_2026]] explicitly notes post-quantum encryption requirements and the need for hardware acceleration for SIKE and SIDH, while [[mishra_2025]] provides a supply-chain and code-transparency context that reinforces the need for migration planning. This makes the roadmap requirement directly traceable without a new gap.

## Acceptance Criteria
- [ ] The system has a documented migration roadmap for post-quantum cryptography.
- [ ] The roadmap identifies which cryptographic components are in scope for future migration.
- [ ] The roadmap can be reviewed as part of an architecture or security readiness assessment.

## Notes
This requirement is a planning and governance requirement, not an immediate algorithm replacement mandate.
